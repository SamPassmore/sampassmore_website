# Sam Passmore website 

Github repository containing files for the personal website of Sam Passmore. [The website can be found here.](https://www.sampassmore.com/)

## Things to remember

To preview the website run in the terminal
'''quarto preview'''

To render the website run in the terminal
'''quarto render''' or click => Render in the RStudio Toolbar


To make a new post
1. Create a new folder in posts with the naming structure:
2025-11-12-my-new-post

2. Add an index.rmd file to that folder
